Проект був зроблений в IDE Eclipse.
Тест був створений за допомогою JUnit.
Для запуска тестів необхідно додати проект у workspace.
Запустити тест можливо двома способами:
1. Правий клік на тестовому класі test.com.exoplatform.testtask.qname.ParserTest і вибрати Run-As → JUnit Test. Результат тесту буде зображений у вікні JUnit View.
2. Запустити на виконання файл  test.com.exoplatform.testtask.qname.TestMain. Цей клас запустить на виконання тестовий клас ParserTest і в консолі буде виведено кількість пройдених тестів і опис непройдених тестів, якщо такі є.
